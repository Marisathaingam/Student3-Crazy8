#Crazy 8 Rules Summary 

## Game Mode
1 player vs 1 computer

## Initial Cards
Each player receives 5  cards.

## Valid Move 
A card can be played if:
- it matches the current suit, or
- it matches the current rank, or
- it is an 8

## Spacial Card
An 8 can be Played at any time and allows the player to choose a new suit.

## Draw Rule  
If no valid card is available, the player must draw 1 card.

## win Condition
The first participant with no cards remaining wins the game.

## Empty Draw Pile 
When the draw pile is empty, the discard pile is reshuffled except for the top card.
